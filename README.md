# Reality Toolkit - Locomotion

Locomotion System for use with the [Reality Toolkit](https://github.com/realitycollective/com.realitytoolkit.core).

## What's included?

### Teleport Locomotion Providers

- Blink Teleport
- Dash Teleport
- Instant Teleport

### Free Locomotion Providers

- Smooth Locomotion

## Requirements

- [RealityToolkit.Core](https://github.com/realitycollective/com.realitytoolkit.core)

## Getting Started

### OpenUPM

[![openupm](https://img.shields.io/npm/v/com.realitytoolkit.locomotion?label=openupm&registry_uri=https://package.openupm.com)](https://openupm.com/packages/com.realitytoolkit.locomotion/)

The simplest way to get started using the package in your project is via OpenUPM. Visit [OpenUPM](https://openupm.com/docs/) to learn more about it. Once you have the OpenUPM CLI set up use the following command to add the package to your project:

```
`openupm add com.realitytoolkit.locomotion`
```

> For more details on using [OpenUPM CLI, check the docs here](https://github.com/openupm/openupm-cli#installation).
